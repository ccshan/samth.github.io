Research

My research focuses on programming languages that grow with their programs. I have developed Typed Racket, a statically-typed dialect of Racket, allowing existing scripts to be enriched with types.

I am also working on the next revision of the JavaScript programming language, including enabling programmers to evolve existing JavaScript code to use new language features such as modules.

In the past, I've worked on the Fortress language, at Sun Microsystems, a language designed for scientific computing.
