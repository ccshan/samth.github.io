Typed Scheme

Typed Scheme is a typed dialect of PLT Scheme. It integrates with modules written in other PLT dialects, and provides a type system designed to support common Scheme idioms.

Installation

Typed Scheme is available as a package from PLaneT . It can be installed with the following require statement:

> (require (planet "info.ss" ("plt" "typed-scheme.plt")))

DrScheme should then be restarted after installation.

Documentation

The manual provides an introduction and reference for Typed Scheme.

More technical material on Typed Scheme is available from the various papers.
